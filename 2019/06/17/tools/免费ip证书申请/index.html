免费ssl证书申请

2019-06-17 tools

有时候因为业务需求 可能需要使用ssh证书 但是个人开发者很难搞到域名 使用别的域名签发的证书浏览器会报不信任

目前有一家大陆新兴 SSL 证书厂商（环洋诚信）推出免费 3 个月的多域名证书，支持泛域名、单域名、IP 、多域名或其混合，支持续期

  • 打开购买证书

这里点击继续 之后输入账单信息 点击右侧结账


这一步就可以配置证书了 证书支持csr 也支持在线生成 如果自己生成的话需要使用中国数字证书提供的 在线生成工具

在这里 我们使用在线的生成方案


点击后会让你输入省市信息 正常填写就行了

填写完成后 在下面域名列表中输入可用的域名 这一以 example.com 为例

点击下一步后就到了验证阶段，其中，域名支持 DNS、HTTP、HTTPS、邮件验证，IP 仅支持 HTTP、HTTPS 验证

http/https验证方法:下载验证文件，保存到待验证域名的 .well-known/pki-validation/ 目录下，可以自己先访问 http:/ /域名/.well-known/pki-validation/验证文件看看是否可以打开，确认可用后保存提交验证即可。

网站配置没有问题的话，上传文件后就可以保存提交了，基本上秒验证：

验证通过后刷新页面 可以看到证书的下载入口了 其中转换证书可以根据不同的产品(nginx,apache等) 自动生成的证书格式(私钥在邮箱里)


点击后输入公钥和私钥 会下载几个主流服务器的ssl证书的key和pem文件
验证一下

没有报站点不信任 整个
