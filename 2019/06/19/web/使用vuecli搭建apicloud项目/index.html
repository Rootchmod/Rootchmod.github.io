使用vuecli搭建apicloud项目

2019-06-19 web

本次教大家如何使用vue-cli3完美编写apicloud项目，个人也有些问题没解决，但是终归是没报错而且能运行的，
我之前也在网上找了好多，但是他们的回答完全一样，对完全，一模一样，这让我也对国内的人很鄙视

首先我使用的vue-cli3 + typescript 的项目，
项目难度只增不减

首先搭建项目 按照一个正常的vue-cli3项目来搭建就行了

之后是要使用到官方的api接口,每次在新建apicloud项目的时候会有个script文件夹,里面有个api.js文件

把这个文件复制到自己的cli项目中

使用:在import完了之后引入

之后的调用直接使用$api就好了

另外有些地方需要使用api(非$api),我看了一下,apicloud是把这个对象封装到全局里面了,意思就是:只要是使用apicloud运行,
api这个对象都是能调用到的,就相当于浏览器的window对象,api这个对象是和window对象同级的,所以你使用window.api是调用
不到的

这样整合就完成了,但是 会有些地方报错,我的解决方法就是:不解决,编译器报错不影响运行,且运行中不会报错

api会报错/$api会报错/esline会报相关错误/在编译打包的时候也会报找不到之类的相关错误(这里需要注意,只要是成功生成了dist文件
,相关的打包报错都不要理会,不会影响项目正常运行的,js也不会在运行时报错)
